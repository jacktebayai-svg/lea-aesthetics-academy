### Critical Data Mapping

#### User Accounts
```sql
-- Prisma -> Supabase mapping
SELECT 
  id,
  email,
  CASE role 
    WHEN 'ADMIN' THEN 'owner'::user_role
    WHEN 'CLIENT' THEN 'client'::user_role  
    WHEN 'STUDENT' THEN 'student'::user_role
  END as role,
  json_build_object(
    'firstName', first_name,
    'lastName', last_name,
    'phone', phone
  ) as profile
FROM prisma.users;
```
